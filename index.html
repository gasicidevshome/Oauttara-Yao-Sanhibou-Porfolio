A propos de moi

Développeur full-stack web & mobile (22 ans), basé en Côte d’Ivoire — expérience pratique depuis 2020. J’interviens sur l’ensemble du cycle : spécification fonctionnelle, architecture back/front, intégration, tests et déploiement.
Ce que je fournis :


  • Frontend : HTML5, CSS moderne (responsive, animations légères), JavaScript vanilla/ES6 — UI claires et performantes.
  • Backend & APIs : PHP (REST), Python (scripts, automations, petites API), bonnes pratiques sécurité et optimisation.
  • Mobile : Dart + Flutter pour apps cross-platform fluides ; connaissances en Python pour outils/automatisations (Kivy/Tkinter selon besoin).
  • Outils & pratiques : gestion de versions (Git), structuration modulaire du code, documentation, tests basiques et déploiement continu quand demandé.

Exemples d’impact : amélioration de l’UX entraînant +X% de conversion (exemples dans la section projets), automatisation de tâches répétitives via scripts Python, et intégrations API stables pour synchronisation de données multi-sites. Si vous cherchez un développeur pragmatique qui allie autonomie technique et sens produit — parlons de votre projet. Je fournis aussi estimations techniques et prototypes rapides pour valider les choix dès les premières étapes.
